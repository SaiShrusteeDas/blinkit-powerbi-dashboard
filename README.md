# BlinkIT Grocery Sales Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes BlinkIT grocery sales performance.

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling

## KPIs
- Total Sales
- Average Sales
- Number of Items
- Average Rating

## Dashboard Features
- Outlet Size Analysis
- Outlet Location Analysis
- Item Type Analysis
- Fat Content Analysis
- Outlet Type Performance

## Insights
1. Tier 3 outlets generated the highest sales.
2. Regular fat products contributed more sales.
3. Fruits and snacks were among the top-selling categories.
